When running the game: 
	WASD to move the player
	Left mouse button to shoot
	Right mouse to use powerup
Weapons
	1: Fist
	2: Starter Gun
	3: Machine Gun
	4: Knife
	5: Pipe
	6: Bow and Arrow
	7: Rocket Launcher
	8: Sword
ESC opens the crafting menu
E and Q change current buff selected
Scroll wheel changes the weapon selected
Space uses the teleport after activating it with E (Green energy bar is the teleport Icon)
